# 🏥 Clínica Central IMA - Simulador de Turnos

Proyecto Final de JavaScript - CoderHouse

## 📌 Descripción
Simulador interactivo que permite gestionar turnos médicos de forma completa, incluyendo registro de usuarios, inicio de sesión y reserva de turnos.

## 🚀 Funcionalidades
- Registro de pacientes
- Inicio de sesión
- Reserva de turnos
- Cancelación de turnos
- Persistencia con LocalStorage
- Carga de especialidades desde JSON

## 🛠️ Tecnologías
- HTML
- CSS
- JavaScript
- JSON
- SweetAlert2
- Toastify

## 📂 Estructura
- `/css` → estilos
- `/js` → lógica del sistema
- `/data` → archivo JSON con especialidades
- `index.html` → página principal

## ▶️ Cómo ejecutar
Abrir con Live Server o desde GitHub Pages.
